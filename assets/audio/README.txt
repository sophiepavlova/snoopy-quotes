Audio Assets Directory
======================

The app generates typing sounds programmatically using the Web Audio API.

If you'd like to use a custom typing sound file instead:

1. Add your audio file here (e.g., typing.mp3 or typing.wav)
2. Modify the TypewriterSound class in app.js to use the audio file

Supported formats:
- MP3 (best browser support)
- WAV (larger file size)
- OGG (good compression)

For best results:
- Keep files under 100KB
- Use short, crisp typing sounds (50-100ms)
- Normalize audio levels
